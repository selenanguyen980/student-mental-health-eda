This project analyzes survey data from 101 university students to explore the relationship between mental health indicators and academic performance using Python and pandas.
I examined CGPA distribution, treatment seeking behavior, and anxiety rates across year groups to identify patterns between depression, anxiety, and academic outcomes. The analysis focuses on data cleaning, exploratory data analysis, groupby aggregations, and data visualization techniques to translate raw survey data into meaningful findings.

### Key Findings
- Students with depression were more concentrated in the 3.00–3.49 CGPA range, while students without depression peaked in the 3.50–4.00 range
- Most students with depression were not seeking treatment from a specialist
- Anxiety rates remained consistent across all four years of study, ranging from 25% to 38%

### Tools & Skills
Tools: Python (pandas), Tableau, Excel

Skills: Data cleaning, exploratory data analysis, descriptive statistics, groupby aggregations, data visualization
